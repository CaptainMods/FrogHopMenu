## FrogHopMenu
Frog Sqwad Mod Menu

Requires: BepInEx_win_x64_5.4.23.5.zip

Found Here: https://github.com/BepInEx/BepInEx/releases

Step By Step:

1. Download: BepInEx_win_x64_5.4.23.5.zip From https://github.com/BepInEx/BepInEx/releases

2. Go to Steam > Library > Frog Sqwad > Gear Icon (Right Hand Side) > Manage > Browse Local Files

3. Find the .zip and press "Extract All"

4. Put the path as: C:\Program Files (x86)\Steam\steamapps\common\Frog Sqwad Demo

5. Click Extract

6. RUN THE GAME ONCE

7. Then Download: FrogHopMenu.dll From the Realease Channel (Right Side Middle Of This Page)
   or go to here: https://github.com/CaptainMods/FrogHopMenu/releases/tag/ModMenu

9. Drag and Drop it into: C:\Program Files (x86)\Steam\steamapps\common\Frog Sqwad Demo\BepInEx\plugins

10. Click Play and Enjoy

Note: Source Code has all the Code Used in This Project
